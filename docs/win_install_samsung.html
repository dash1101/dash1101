Windows 11 Installation Guide

1. Turn on the laptop and repeatedly press F2 to enter the BIOS.
2. Go to the Boot tab → Boot Options.
3. Select your Samsung Fit USB drive as the first boot device.
4. Save and exit the BIOS.
5. At the Windows installer, click Next.
6. Select Install Now.
7. If asked for a product key, press I don’t have a product key.
8. If prompted to choose a version, select Windows 11 Pro.
9. On “Where do you want to install Windows?”, press Shift + F10 to open Command Prompt.
10. Type each of these one at a time:
  • diskpart
  • list disk
Send me the list so I can confirm which disk to choose.
11. After I confirm the correct disk, type:
  • sel dis #
  • clean
12. Close Command Prompt and press Refresh.
13. Select the unallocated space and click Next.
14. Windows will now install automatically.
The laptop may restart several times — totally normal.
